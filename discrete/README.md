# Discrete Mathematics Course Materials

This repository contains materials for a Discrete Mathematics course, including lectures and practical exercises in Armenian.

## Lectures (Դասախոսություններ)

| File | Topic (Armenian) | Topic (English) | Description |
|------|------------------|-----------------|-------------|
| [Lecture 1](Դասախոսություններ/lecture1-discrete.pdf) | Բազմությունների տեսություն | Set Theory | Introduction to sets, elements, subsets, and set equality. |
| [Lecture 2](Դասախոսություններ/lecture2-discrete.pdf) | Հարաբերություններ | Relations | Binary relations, Cartesian products, ordered pairs. |
| [Lecture 3](Դասախոսություններ/lecture3-discrete.pdf) | Մասնակի կարգ | Partial Orders | Partial order relations, reflexive, antisymmetric, and transitive properties. |
| [Lecture 4](Դասախոսություններ/lecture4-discrete.pdf) | Արտապատկերումներ | Functions/Mappings | Functions as relations, domain, codomain, images. |
| [Lecture 5](Դասախոսություններ/lecture5-discrete.pdf) | Կոմբինատորիկա | Combinatorics | Basic counting principles, sum and product rules. |
| [Lecture 6](Դասախոսություններ/lecture6-discrete.pdf) | Զուգորդություններ | Combinations | Binomial coefficients, combinations vs permutations. |
| [Lecture 7](Դասախոսություններ/lecture7-discrete.pdf) | Տրոհումներ | Partitions | Set partitions and their relation to mappings. |
| [Lecture 8](Դասախոսություններ/lecture8-discrete.pdf) | Ռեկուրենտ առնչություններ | Recurrence Relations | Solving recurrence usage in counting problems (e.g., frogs problem). |
| [Lecture 9](Դասախոսություններ/lecture9-discrete.pdf) | Դիսկրետ ֆունկցիաներ | Discrete Functions | Boolean functions, logic. |
| [Lecture 10](Դասախոսություններ/lecture10-discrete.pdf) | Ժեգալկինի բազմանդամ | Zhegalkin Polynomials | Algebraic normal form of Boolean functions. |
| [Lecture 12](Դասախոսություններ/lecture12-discrete.pdf) | Գրաֆների տեսություն | Graph Theory | Definitions of graphs, vertices, edges, multigraphs. |
| [Lecture 13](Դասախոսություններ/lecture13-discrete.pdf) | Հարթ գրաֆներ | Planar Graphs | Planarity, Euler's formula, planar representations of graphs. |

## Practical Exercises (Առաջադրանքներ)

| [Practice 2-3](Առաջադրանքներ/practice2-3-discrete.pdf) | Հարաբերություններ | Relations | Problems on properties of relations (reflexive, symmetric, transitive). |

| [Practice 5](Առաջադրանքներ/practice5-discrete.pdf) | Կոմբինատորիկա | Combinatorics | Basic counting problems (sum/product rules), divisibility. |

| [Practice 8](Առաջադրանքներ/practice8-discrete.pdf) | Ռեկուրենտ առնչություններ | Recurrence Relations | Solving linear recurrence relations. |
