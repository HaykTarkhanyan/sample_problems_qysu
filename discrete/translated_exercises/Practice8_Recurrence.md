# Practice 8: Recurrence Relations

1. Given $a_1 = 3, a_2 = 9$. Find $a_{200}$ if the recurrence relation is $a_{n+2} = a_{n+1} - a_n$.

2. Solve the linear recurrence relation:
   $a_n = 5a_{n-1} - 6a_{n-2}$, for $n > 2$.
   Initial conditions: $a_1 = 2, a_2 = 10$.

3. Solve the linear recurrence relation:
   $a_n = 4a_{n-1} - 4a_{n-2}$, for $n \ge 2$.
   Initial conditions: $a_0 = 2, a_1 = 6$.

4. Solve the linear recurrence relation:
   $a_n = 2\sqrt{2}a_{n-1} - 4a_{n-2}$, for $n \ge 2$.
   Initial conditions: $a_0 = 1, a_1 = 2$.

5. Solve the linear recurrence relation:
   $a_n = 2a_{n-1} + a_{n-2}$, for $n \ge 2$ (Fibonacci-like).
   Initial conditions: $a_0 = 0, a_1 = 1$.

6. How many binary strings of length 10 are there that contain two consecutive zeros? (Complementary count: total - no consecutive zeros).

7. How many words can be formed from letters $\{ a, b, c \}$ such that $a$ and $b$ are adjacent?
